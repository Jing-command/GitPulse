# GitPulse 前端设计规范

## 一、设计原则

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| 一致性 | 统一的视觉语言和交互模式 |
| 简洁性 | 减少视觉噪音，突出核心内容 |
| 可访问性 | 符合 WCAG 2.1 AA 标准 |
| 响应式 | 适配多种屏幕尺寸 |
| 性能优先 | 优化加载速度和交互响应 |

### 1.2 设计系统

基于 **shadcn/ui** + **Tailwind CSS** 构建设计系统，确保组件可定制性和开发效率。

## 七、组件规范

### 7.1 按钮

#### 类型

| 类型 | 样式 | 用途 |
|------|------|------|
| Primary | 蓝色背景 | 主要操作 |
| Secondary | 灰色背景 | 次要操作 |
| Outline | 边框样式 | 辅助操作 |
| Ghost | 无背景 | 文字链接 |
| Destructive | 红色背景 | 危险操作 |

#### 尺寸

| 尺寸 | 高度 | 字号 | 内间距 |
|------|------|------|--------|
| sm | 32px | 14px | 12px 8px |
| md | 40px | 14px | 16px 8px |
| lg | 48px | 16px | 24px 12px |

#### 状态

- Default：默认状态
- Hover：背景加深 10%
- Active：背景加深 20%
- Disabled：透明度 50%，禁用点击
- Loading：显示加载动画

```tsx
// 按钮组件示例
<Button variant="primary" size="md">
  创建项目
</Button>

<Button variant="outline" size="sm">
  取消
</Button>

<Button variant="destructive" size="md">
  删除
</Button>
```

### 7.2 输入框

#### 类型

| 类型 | 用途 |
|------|------|
| Text | 文本输入 |
| Password | 密码输入 |
| Email | 邮箱输入 |
| Number | 数字输入 |
| Textarea | 多行文本 |
| Search | 搜索输入 |

#### 尺寸

| 尺寸 | 高度 | 字号 |
|------|------|------|
| sm | 32px | 14px |
| md | 40px | 14px |
| lg | 48px | 16px |

#### 状态

- Default：默认边框
- Focus：蓝色边框 + 蓝色阴影
- Error：红色边框 + 错误提示
- Disabled：灰色背景，禁用输入

```tsx
// 输入框组件示例
<Input
  label="项目名称"
  placeholder="请输入项目名称"
  error="项目名称不能为空"
/>

<Textarea
  label="项目描述"
  placeholder="请输入项目描述"
  rows={4}
/>
```

### 7.3 选择器

#### 下拉选择

```tsx
<Select
  label="内容类型"
  options={[
    { value: 'changelog', label: '更新日志' },
    { value: 'technical', label: '技术博客' },
    { value: 'seo', label: 'SEO 文章' },
  ]}
  placeholder="请选择类型"
/>
```

#### 多选

```tsx
<MultiSelect
  label="语言"
  options={[
    { value: 'zh', label: '中文' },
    { value: 'en', label: 'English' },
  ]}
  placeholder="请选择语言"
/>
```

### 7.4 卡片

```tsx
<Card>
  <CardHeader>
    <CardTitle>项目名称</CardTitle>
    <CardDescription>项目描述</CardDescription>
  </CardHeader>

  <CardFooter>

  </CardFooter>

```

### 7.5 表格

```tsx
<Table>
  <TableHeader>
    <TableRow>
      <TableHead>标题</TableHead>
      <TableHead>类型</TableHead>
      <TableHead>状态</TableHead>
      <TableHead>创建时间</TableHead>
      <TableHead>操作</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>v1.2.0 更新日志</TableCell>
      <TableCell><Badge>Changelog</Badge></TableCell>
      <TableCell><Badge variant="success">已发布</Badge></TableCell>
      <TableCell>2024-01-01</TableCell>
      <TableCell>
        <Button variant="ghost" size="sm">编辑</Button>
        <Button variant="ghost" size="sm">删除</Button>
      </TableCell>
    </TableRow>
  </TableBody>
</Table>
```

### 7.6 标签 (Badge)

| 变体 | 用途 |
|------|------|
| default | 默认标签 |
| secondary | 次要标签 |
| success | 成功状态 |
| warning | 警告状态 |
| error | 错误状态 |
| outline | 边框样式 |

```tsx
<Badge>草稿</Badge>
<Badge variant="success">已发布</Badge>
<Badge variant="warning">待审核</Badge>
```

### 7.7 模态框

```tsx
<Dialog open={open} onOpenChange={setOpen}>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>确认删除</DialogTitle>
      <DialogDescription>
        此操作不可撤销，确定要删除吗？
      </DialogDescription>
    </DialogHeader>
    <DialogFooter>
      <Button variant="outline">取消</Button>
      <Button variant="destructive">删除</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

### 7.8 通知 (Toast)

```tsx
// 成功通知
toast.success('操作成功');

// 错误通知
toast.error('操作失败，请重试');

// 警告通知
toast.warning('请注意，此操作有风险');

// 信息通知
toast.info('新版本已发布');
```

## 九、交互规范

### 9.1 动画时长

| 类型 | 时长 | 用途 |
|------|------|------|
| fast | 150ms | 按钮状态变化 |
| normal | 200ms | 下拉展开、模态框 |
| slow | 300ms | 页面切换、大动画 |

### 9.2 缓动函数

| 名称 | 值 | 用途 |
|------|-----|------|
| ease | `cubic-bezier(0.4, 0, 0.2, 1)` | 默认缓动 |
| ease-in | `cubic-bezier(0.4, 0, 1, 1)` | 进入动画 |
| ease-out | `cubic-bezier(0, 0, 0.2, 1)` | 退出动画 |
| ease-in-out | `cubic-bezier(0.4, 0, 0.2, 1)` | 双向动画 |

### 9.3 悬停效果

- 按钮：背景色加深 10%
- 卡片：显示阴影
- 链接：显示下划线
- 图标：颜色加深

### 9.4 加载状态

- 按钮：显示 Spinner
- 页面：显示骨架屏
- 列表：显示加载占位

## 十三、错误处理

### 13.1 表单错误

- 字段下方显示错误信息
- 错误字段红色边框
- 提交时滚动到第一个错误

### 13.2 页面错误

- 404：页面不存在
- 500：服务器错误
- 网络错误：网络连接失败

### 13.3 错误提示

```tsx
<Alert variant="error">
  <AlertTitle>操作失败</AlertTitle>
  <AlertDescription>
    请检查您的输入并重试
  </AlertDescription>
</Alert>
```

## 十四、可访问性规范

### 14.1 颜色对比度

- 正文文字：对比度 ≥ 4.5:1
- 大标题：对比度 ≥ 3:1
- 禁用状态：对比度 ≥ 3:1

### 14.2 键盘导航

- 所有交互元素可通过 Tab 键访问
- 支持快捷键操作
- 焦点状态清晰可见

### 14.3 屏幕阅读器

- 图片提供 alt 属性
- 按钮提供 aria-label
- 表单提供 label 关联
